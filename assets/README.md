# This directory contains image assets for the README and documentation.

## Contents

- `logo.png` - Project logo (120x120px recommended)
- `screenshots/` - UI screenshots for documentation

## Adding Screenshots

To add screenshots:

1. Run the notebook
2. Take screenshots of key features
3. Save as PNG with descriptive names
4. Recommended size: 800-1200px width
